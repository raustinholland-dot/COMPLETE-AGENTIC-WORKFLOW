Zone Legend

Center Circle: Size = deal value. Color = stage. Number = P2V2C2 total. Border animation = trend.
PE Sponsor Halo: Green solid (4-5), amber solid (3), red dashed (1-2), absent (0).
Complexity Ring: 5 arcs: stakeholders, workstreams, doc volume, doc types, calendar. Blue=simple, amber=moderate, red=complex.
Radial Arms: 6 P2V2C2 dimensions. Rings = time buckets (weekly/biweekly/monthly). Width = score. Color = red to green.
Dormant Bands: Gray dashed rings spanning full circle. No scoring activity in that time period.
DAP Boundary: Outer ring. Solid green = agreed. Amber = exists. Dashed = none. Arc = milestones complete.
Stakeholder Dots: Champion (30deg), ES (90deg), DM (210deg). Solid = confirmed. Hollow = potential.
Outer Glow: Intensity and radius scale with deal value. Atmospheric halo effect.
